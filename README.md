# vim-docker

Docker plugin for Vim.

## Usage

### Docker commands

    :Docker <command>

e.g. to list all containers:

    :Docker ps -a

### Docker Machine commands

    :Dmachine <command>

e.g. to list all machines:

    :Dmachine ls


### Docker Compose commands

    :Dcompose <command>

e.g. to list all containers in current Compose project:

    :Dcompose ps
